# 学习报告下载中心 PRD 片段

## 背景

为了降低家长重复咨询，产品希望在教练工作台新增“学习报告下载中心”。老师可以查看学生已生成报告并下载，家长端暂不开放。

## 目标

- 老师可以按学生、时间范围筛选报告
- 老师可以下载 PDF 报告
- 当报告还在生成中时，页面要展示“生成中”状态

## 角色

- 老师
- 学生
- 系统任务服务

## 页面草图说明

页面标题为“学习报告下载中心”，顶部有学生姓名搜索框、时间范围选择器、状态筛选下拉框。列表区域包含：

- 学生姓名
- 报告类型
- 生成时间
- 状态
- 操作按钮

当状态为“已生成”时，操作按钮显示“下载”。
当状态为“生成中”时，操作按钮置灰，不可点击。

## 补充说明

- 文字中写明只有老师可操作
- 但原型截图右上角出现了“家长视角切换”入口，未解释其用途
- 列表最多展示最近 90 天的报告
- 下载链接有效期未说明

## 流程图说明

1. 老师进入页面
2. 设置筛选条件
3. 系统返回报告列表
4. 如果报告状态为已生成，则可点击下载
5. 如果报告状态为生成中，则不可下载

## 时序说明

- 老师发起列表查询
- 页面请求报告服务
- 报告服务返回列表和状态
- 老师点击下载
- 页面请求下载地址
- 报告服务返回下载地址

## 表格

| 状态值 | 展示文案 | 允许下载 |
| --- | --- | --- |
| READY | 已生成 | 是 |
| PROCESSING | 生成中 | 否 |
| FAILED | 生成失败 | 未说明 |

## 验收

- 老师可以看到自己名下学生的报告列表
- 已生成状态的报告可以下载
- 生成中的报告不可点击下载
